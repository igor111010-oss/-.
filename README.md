# 🚀 Space Missions Tracker API

A RESTful API for tracking space missions, astronauts, and rockets.

## Features
- 📋 CRUD operations for space missions
- 👨‍🚀 Manage astronauts and assign them to missions
- 🚀 Track rockets and their mission assignments
- 📊 SQLite database with SQLAlchemy ORM
- 🔍 Advanced filtering and pagination
